What is the difference between block tags and inline tags?
Ans:-Block tags start on a new line and take up the full width available, while inline tags do not start on a new line and only take up as much width as necessary.

What is the margin, border, and padding?
Ans:-Margin is the space outside the border of an element, border is the line surrounding the padding and content, and padding is the space between the content and the border.

What are units of measurement you can use in CSS?
Ans:-Pixels (px), em, rem, percentages (%), viewport width (vw), and viewport height (vh).

When would you use a figure tag?
Ans:-The figure tag is used to encapsulate media content like images, illustrations, diagrams, or code snippets along with their captions, providing semantic meaning and better accessibility.
