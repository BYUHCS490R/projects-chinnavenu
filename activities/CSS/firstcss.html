Explain the difference between inline, embedded, and external CSS?.

Inline CSS – Styles are written directly in the element itself; affects only that one element.
Embedded CSS – Styles are written inside the head section of the HTML file; affect only that page.
External CSS – Styles are written in a separate .css file; can be linked to multiple HTML files, allowing for consistent styling across an entire website.

Explain why you should only use external CSS.

Using external CSS is the best practice because it keeps the design separate from the content, making code cleaner and easier to manage. It allows one stylesheet to be reused across multiple pages, ensuring a consistent look and reducing repetition. Updates are simple, since changing one file applies the style everywhere, and browsers can cache the CSS for faster loading. This makes websites more efficient, professional, and easier to maintain

What is a selector used for?

A selector is used in CSS to target specific HTML elements and apply styles to them. It tells the browser which elements should get the defined styles. For example, if you use a selector for paragraphs, all paragraph elements will follow the styles you set. In short, a selector connects the CSS rules to the HTML elements.
