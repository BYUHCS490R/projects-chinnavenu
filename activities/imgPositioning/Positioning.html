Positioning

1. What is normal flow in HTML? Normal flow is the default way elements are arranged on a web page. Block elements stack vertically one after another, while inline elements flow horizontally within a line. Itâ€™s how the page is displayed before any CSS changes are applied.

2. What does float do? Why do you need to clear the float? Float moves an element to the left or right so that other content can wrap around it.You need to clear the float because floated elements are removed from normal flow, which can cause parent containers to collapse. Clearing ensures following content or containers appear correctly below floated elements.

3. What do absolute, fixed, and relative do? Relative positions an element relative to its normal position. It still occupies space in the layout.Absolute positions an element relative to its nearest positioned ancestor and removes it from normal flow.Fixed positions an element relative to the browser window so it stays in place even when scrolling.
