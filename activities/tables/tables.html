When do you use a table?

Use a table when you need to display data or information in rows and columns for easy comparison (like schedules, prices, or statistics)

When do you not use a table?

Do not use a table for page layout or design purposes — use CSS for that instead.

When do you need to do extra work for table accessibility?

Do extra accessibility work when your table has complex structures (like merged cells or multiple headers) — add , scope, and captions to help screen readers.

What is the easiest way to color every other row a different color?

The easiest way is to use CSS with the nth-child
