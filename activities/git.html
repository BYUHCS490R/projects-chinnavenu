Describe the tags we discussed today. Remember, you can look at the lecture notes for a video that reviews what we discussed.

Doctype = tells the computer this is an HTML document.

HTML = the container for the whole page. Head = holds information about the page, like the title and settings, but not the content people see. Title = the name that shows up on the browser tab. Body = everything you actually see on the page, like text, images, and links. Headings = titles or subtitles, from biggest (heading one) to smallest (heading six). Paragraph = used for blocks of text. Link = makes clickable text that can take you to another page. Image = adds a picture to the page. Lists = can be ordered (numbered) or unordered (bulleted), and each item in the list is marked separately.
